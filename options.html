Redirect Click'n'Load

Settings

How it works

This Extension changes the Hostname/IP of requests to localhost:96666/127.0.0.1:9666. The value you entered above will be set as new Hostname/IP.

With this Extension you don't need to setup an SSH Port Forward anymore.
